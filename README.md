Personal Health Monitor

Project Overview

Personal Health Monitor is a Python-based console application developed to help users monitor and manage their personal health. The application provides essential health assessment tools through a simple menu-driven interface, allowing users to calculate their Body Mass Index (BMI), track daily water intake, monitor calorie consumption, and access blood group compatibility information.

Objectives

The main objectives of this project are to:

* Promote healthy lifestyle habits.
* Help users monitor important health indicators.
* Provide quick access to blood group compatibility information.
* Practice Python programming by developing a real-world application.

Features

1. BMI Calculator

* Accepts the user's weight and height.
* Calculates Body Mass Index (BMI).
* Displays the BMI value.
* Classifies the BMI as:

  * Underweight
  * Normal Weight
  * Overweight
  * Obesity

2. Water Intake Tracker

* Calculates the recommended daily water intake based on body weight.
* Allows users to record the amount of water consumed.
* Displays:

  * Recommended intake
  * Water consumed
  * Remaining water
  * Daily progress percentage

3. Calories Counter

* Records calories consumed during:

  * Breakfast
  * Lunch
  * Dinner
  * Snacks
* Calculates total daily calorie intake.
* Classifies calorie intake as Low, Healthy, or High.

4. Blood Information

* Accepts the user's blood group.
* Displays compatible blood donors.
* Displays compatible blood recipients.

Technologies Used

* Programming Language: Python
* Development Environment: Visual Studio Code
* Programming Concepts:

  * Variables
  * Input and Output
  * Conditional Statements
  * Loops
  * Mathematical Operations
  * String Formatting

Program Flow

1. Display the main menu.
2. Allow the user to select a module.
3. Execute the selected health tool.
4. Display the results.
5. Return to the main menu.
6. Exit only when the user selects the Exit option.
